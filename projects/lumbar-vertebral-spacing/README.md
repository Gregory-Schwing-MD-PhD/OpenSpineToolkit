# Lumbar Vertebral Spacing

**Miniproject — OpenSpineToolkit**

## Goal
Measure the spacing between adjacent vertebral body centroids along the lumbar
spine (and its variation per level).

## How CTSpinoPelvic1K v3 helps
The lumbar bodies (L1–L6) were already GT, but v3's **thoracic GT (T1–T13)** lets
you extend the spacing profile **through the thoracolumbar junction**, giving
normal-range context above the lumbar spine and a cleaner reference for detecting
abnormal lumbar spacing.
