# Scoliosis / Cobb Angle

**Miniproject — OpenSpineToolbox**

## Goal
Measure the coronal Cobb angle from the tilt of the end vertebrae of a curve.

## How CTSpinoPelvic1K v3 helps
The Cobb angle is measured between the most-tilted **end vertebrae**, which for
most curves are **thoracic or thoracolumbar** — not lumbar. v3 ships full
**thoracic ground truth (T1–T13)**, so you can identify end vertebrae above L1
instead of being capped at the lumbar spine. The **rib** labels give an
independent rib-bearing-level check to confirm vertebral identity.
