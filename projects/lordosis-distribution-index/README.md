# Lordosis Distribution Index (LDI)

**Miniproject — OpenSpineToolkit**

## Goal
Compute LDI = (lower-arc lordosis / total lumbar lordosis) × 100, describing how
lordosis is apportioned across the lumbar spine.

## How CTSpinoPelvic1K v3 helps
LDI needs the **full lordotic arc** and a stable upper bound. v3's **thoracic GT**
defines the thoracolumbar inflection / upper reference, and the **femur GT**
provides the pelvic landmarks used to normalise and contextualise the arc
(relating LDI to pelvic incidence).
