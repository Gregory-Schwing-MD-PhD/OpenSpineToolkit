# Spinal Stenosis

**Miniproject — OpenSpineToolbox**

## Goal
Detect and grade narrowing of the spinal canal at each level.

## How CTSpinoPelvic1K v3 helps
Reliable **per-level localisation** is half the problem. v3's **thoracic GT** plus
the lumbar/sacral column give a robust level index for enumerating vertebrae
correctly — especially important in transitional anatomy (LSTV), where miscounting
levels is the classic failure mode.

> Rib labels (rib-bearing vs non-rib-bearing is an extra enumeration cue) are
> **deferred to v4** — use the vertebral column for now.
