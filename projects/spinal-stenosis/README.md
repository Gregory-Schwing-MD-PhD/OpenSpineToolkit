# Spinal Stenosis

**Miniproject — OpenSpineToolbox**

## Goal
Detect and grade narrowing of the spinal canal at each level.

## How CTSpinoPelvic1K v3 helps
Reliable **per-level localisation** is half the problem. v3's **thoracic GT**
plus the **rib labels** give a robust level index: rib-bearing vs non-rib-bearing
status helps enumerate vertebrae correctly — especially important in transitional
anatomy (LSTV), where miscounting levels is the classic failure mode.
