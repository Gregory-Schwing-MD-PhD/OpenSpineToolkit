# Vertebral Body Wedging Index

**Miniproject — OpenSpineToolbox**

## Goal
Compute the wedging index (anterior vs posterior vertebral body height ratio) per
vertebra, a marker of compression fracture and structural kyphosis/scoliosis.

## How CTSpinoPelvic1K v3 helps
Wedging is most diagnostic in the **thoracic spine** (kyphosis, compression
fractures) — exactly the region earlier versions didn't label. v3's **thoracic GT
(T1–T13)** now provides those bodies, and the **rib** labels help confirm thoracic
level identity so you index the wedged vertebra correctly.
