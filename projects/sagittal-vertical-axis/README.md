# Sagittal Vertical Axis (SVA)

**Miniproject — OpenSpineToolkit**

## Goal
Measure SVA: the horizontal offset between the C7/T1 plumb line and the
posterosuperior corner of S1.

## How CTSpinoPelvic1K v3 helps
v3 gives you the **caudal anchor** reliably from labels — the sacrum
(posterosuperior S1 corner) plus the femoral heads for pelvic context — and adds
thoracic vertebrae as a rostral reference where they're in view.

> **FOV caveat (important):** classic SVA drops a plumb line from **C7/T1**, but
> these spinopelvic scans usually only reach down from about **T8**, so C7/T1 is
> typically out of view. Expect to compute true SVA only on cases with sufficient
> cranial FOV, or define a thoracolumbar-level variant from the most cranial
> labelled vertebra and state it explicitly. Reporting per-case cranial coverage
> is a useful first step.
