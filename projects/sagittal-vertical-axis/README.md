# Sagittal Vertical Axis (SVA)

**Miniproject — OpenSpineToolbox**

## Goal
Measure SVA: the horizontal offset between the C7/T1 plumb line and the
posterosuperior corner of S1.

## How CTSpinoPelvic1K v3 helps
SVA needs a **rostral landmark** that earlier versions didn't have — the column
stopped at L1, so there was no upper anchor. v3's **thoracic GT provides T1**,
and the **sacrum GT** gives the posterosuperior S1 corner, so you now have *both*
ends of the plumb-line measurement directly from labels.
