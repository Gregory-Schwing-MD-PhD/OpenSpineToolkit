# Lordosis — TRALL Angle

**Miniproject — OpenSpineToolbox**

## Goal
Quantify lumbar lordosis (e.g. the TRALL / total & segmental lordotic angles)
from vertebral endplate orientations.

## How CTSpinoPelvic1K v3 helps
Lordosis is bounded superiorly at the **thoracolumbar junction**. v3's
**thoracic GT (especially the T12 endplate)** gives a stable rostral reference
for the upper arm of the angle, instead of having to assume it from the L1 body
alone. Pair with the sacrum/femur landmarks for the pelvic end.
