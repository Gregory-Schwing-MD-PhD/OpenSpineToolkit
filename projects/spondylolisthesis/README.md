# Spondylolisthesis

**Miniproject — OpenSpineToolkit**

## Goal
Detect and grade vertebral slip (e.g. Meyerding grade) from the AP offset of
adjacent vertebral bodies.

## How CTSpinoPelvic1K v3 helps
Slip is measured between **adjacent labelled bodies**. v3 extends the labelled
column into the **thoracic spine (T1–T13)**, so more adjacent-level pairs are
measurable and you have a cleaner posterior-margin / centroid reference above the
lumbar spine for the offset computation.
