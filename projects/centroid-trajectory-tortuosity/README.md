# Centroid Trajectory Analysis / Tortuosity

**Miniproject — OpenSpineToolbox**

## Goal
Trace the 3D trajectory of vertebral body centroids and quantify spinal
tortuosity / curvature along it.

## How CTSpinoPelvic1K v3 helps
A longer labelled column yields a far richer trajectory. v3's **thoracic GT
(T1–T13)** extends the centroid chain from the lumbar spine up through T1, so you
get the **full thoracolumbar curve** to characterise tortuosity, instead of a
short lumbar-only segment.
