# Centroid Trajectory Analysis / Tortuosity

**Miniproject — OpenSpineToolkit**

## Goal
Trace the 3D trajectory of vertebral body centroids and quantify spinal
tortuosity / curvature along it.

## How CTSpinoPelvic1K v3 helps
A longer labelled column yields a far richer trajectory. v3's **thoracic GT**
extends the centroid chain from the lumbar spine up through the **FOV-visible
thoracic vertebrae** (typically to about **T8**, FOV-permitting), so you get the
**thoracolumbar curve** to characterise tortuosity instead of a short lumbar-only
segment.
