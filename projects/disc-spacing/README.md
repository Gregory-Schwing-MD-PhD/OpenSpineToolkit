# Disc Spacing

**Miniproject — OpenSpineToolkit**

## Goal
Estimate intervertebral disc height from the gap between adjacent vertebral
body endplates.

## How CTSpinoPelvic1K v3 helps
A disc space needs **both** bounding vertebrae labelled. v3 extends the labelled
bodies up the **thoracic column (T1–T13)**, so you can measure thoracic and
thoracolumbar disc spaces — not just L1–S1 — and build a full disc-height profile
along the spine.
